# 🎓 The Batch Vault | 2022-2026
**The Ultimate College Memory Portal**

## 🚀 Overview

**The Batch Vault** is an interactive, fun-loaded digital time capsule built exclusively for our graduating batch. It acts as a centralized digital hub hosting 50+ nostalgic modules—ranging from inside jokes and mini-games to visual galleries and digital graffiti walls. Designed to be purely engaging and nostalgia-driven, it perfectly captures the essence of 4 years of college life without any complex databases or sensitive personal information.

## ✨ Key Features

* 📸 **The Evolution Slider:** Interactive face transformations showcasing the journey from 1st-year innocents to Final-year legends.
* 🖊️ **The Missing Pen Gallery:** A humorous visual exhibition dedicated to all the pens mysteriously lost over 8 semesters.
* ☕ **The Canteen Run:** A lightweight 2D interactive mini-game to escape proxy checks and collect virtual samosas.
* 🔊 **Classic Staff Hits:** A dedicated soundboard featuring the most legendary and repeated dialogues from our professors.
* ✍️ **Digital Last Bench:** A real-time virtual graffiti wall where batchmates can leave their classic quotes and inside jokes.
* 💎 **Premium UI/UX:** Built with modern CSS variables, clean Glassmorphism elements, fluid keyframe animations, and a highly responsive grid architecture.

## 🛠️ Tech Stack

**Frontend**
* HTML5 & Semantic Markup
* CSS3 (Glassmorphism, CSS Grid, Custom Animations)
* Vanilla JavaScript (ES6+, DOM Manipulation, Dynamic Rendering)
* Google Fonts ('Poppins') & Native OS Emojis (for zero-lag loading)

**Architecture**
* **Dynamic Hub System:** A single, lightweight `main.js` engine that renders 50+ features dynamically into the main dashboard without hardcoding HTML.

## 💻 Running the Project Locally

### 📦 Installation

Since this project utilizes a clean, serverless static architecture, setting it up locally is incredibly fast. Follow these steps:

**1️⃣ Clone the repository**
```bash
git clone [https://github.com/nirmalbabu05/The-Batch-Vault.git](https://github.com/yourusername/The-Batch-Vault.git)
cd The-Batch-Vault
```

**2️⃣ Open the project in your favorite editor**
```bash
code .
```

**3️⃣ Start a local development server**
If you are using VS Code, simply install the **Live Server** extension and click "Go Live" on `index.html`. Alternatively, use Python:
```bash
python -m http.server 8000
```

**4️⃣ Open the project in your browser at:**
```text
http://localhost:8000/
```
